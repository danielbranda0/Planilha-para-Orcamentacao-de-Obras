# Planilha para Orçamentação de Obras
Planilha para orçamentação de obras privadas, com quantitativos por serviço e seleção de itens de composições pré-cadastradas. Permite editar índices e preços, gera folha de orçamento para clientes, calcula NF, curva ABC de insumos e serviços, entre outros relatórios.
